# HR-система управления сотрудниками (КЭТ им. Ф.В. Чижова)

Данный проект представляет собой **полностековое веб-приложение** для автоматизации работы отдела кадров. Разработан в рамках курсовой работы по дисциплине «Проектирование и дизайн информационных систем».

##  Функциональность

- **Просмотр списка сотрудников** – таблица с ФИО, должностью, отделом.
- **Фильтрация по отделам** – динамическое обновление списка.
- **Добавление сотрудника** – форма с валидацией, сохранение в БД.
- **Редактирование сотрудника** – модальное окно, обновление данных.
- **Удаление сотрудника** – с подтверждением.
- **Сохранение данных** – все изменения сохраняются в PostgreSQL.

##  Технологии

### Бэкенд
- **Python 3.11+**
- **FastAPI** – создание REST API
- **SQLAlchemy** – ORM для работы с PostgreSQL
- **Pydantic** – валидация данных
- **PostgreSQL** – реляционная база данных
- **JWT** – аутентификация (базовая)
- **Uvicorn** – ASGI-сервер

### Фронтенд
- **React 18** – библиотека для UI
- **Vite** – сборка и разработка
- **Axios** – HTTP-запросы к API
- **CSS** – адаптивная стилизация

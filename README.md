# HR-система (учебный проект)

Данный проект представляет собой простое веб-приложение для управления списком сотрудников отдела кадров. Разработан в рамках изучения основ React и курсовой работы по дисциплине «Проектирование и дизайн информационных систем».

##  Функциональность

- Просмотр списка сотрудников в таблице.
- Отображение информации: ФИО, должность, отдел.
- Фильтрация списка по отделам (в коде заложена, но пока статична).
- Возможность добавления новых сотрудников (реализована форма, но для полной функциональности требуется доработка бэкенда).

##  Технологии

- **React** — библиотека для построения пользовательских интерфейсов.
- **Vite** — быстрый инструмент сборки и разработки.
- **JavaScript (ES6+)** — язык программирования.
- **CSS** — для базовой стилизации (можно расширить).
